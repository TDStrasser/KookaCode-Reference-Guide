**KookaCode** is a powerful script editor designed for the **Kookaberry** and related microprocessors. 
This editor runs within a web browser as a Progressive Web Application (PWA).  

This document describes how to use to the **KookaCode**  script editing tool.

**KookaCode** and the **KookaCode** script editing tool were commissioned by the AustSTEM Foundation 
and created by Damien George for the **Kookaberry**.

This guide is for **KookaCode** v1.0.
